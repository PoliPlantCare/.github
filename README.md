# PlantCare

O PlantCare é um aplicativo e um dispositivo IoT que te ajuda a cuidar das suas plantas! O projeto é desenvolvido para a matéria PCS3100 (Introdução à Engenharia da Computação) da Escola Politécnica da Universidade de São Paulo pelos integrantes Eduardo, Giulia, David e Júlia! Essa organização no GitHub reúnes os repositórios para o servidor, aplicativo e firmware desse projeto.
